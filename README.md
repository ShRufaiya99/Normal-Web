# 🌐 Personal Portfolio Website

Welcome to my personal portfolio! This project is a reflection of my growth and skills as a **Frontend Developer**. It serves as a digital resume to showcase my technical expertise and the projects I've built.

### 🚀 Features:
- **About Me:** A brief introduction to my background and goals.
- **Skills Section:** Highlighting my core technologies (HTML, CSS, JS).
- **Project Gallery:** Showcasing my best work with live links.
- **Contact Form:** A way for recruiters and collaborators to reach me.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop views.

### 🛠️ Technologies Used:
- **HTML5:** For clean and semantic structure.
- **CSS3:** For custom styling, Flexbox layouts, and animations.
- **Google Fonts & FontAwesome:** For typography and icons.
